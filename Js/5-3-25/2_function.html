1. Function declaration

2. Function expression

3. arrow function

1. Function declaration

        function function_name(parameters)
            //code
    }
    
        call function_name(argument)

2. function expression

        const function_name = function(parameters){
            //code
        }

3. Arrow function

        const function_name = (parameters)=>{
            //code
        }
        //callfunction_name(arg)
